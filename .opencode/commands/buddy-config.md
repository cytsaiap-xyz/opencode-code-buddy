description: "⚙️ View or update LLM configuration"
---

Use the `buddy_config` tool to view or update LLM configuration.

Actions available:

- `view` - Show current configuration
- `set_api_key` - Set OpenAI-compatible API key
- `set_model` - Set model name (e.g., gpt-4o, gpt-4o-mini)
- `set_base_url` - Set custom API base URL
